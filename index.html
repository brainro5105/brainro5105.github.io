💡 What is org-pad.el?

org-pad.el links your iPad or tablet to Emacs. It lets you draw with a stylus on your tablet and see the result in your org-mode documents. You do not need to scan files or move images manually. The software handles the transport of your sketches into your project files automatically through your web browser.

📥 Getting Started

Follow these steps to set up the software on your Windows machine.

  1. Visit the official releases page to download the latest version for Windows.
  2. Locate the downloaded file in your downloads folder.
  3. Double-click the installer icon to start the setup process.
  4. Follow the on-screen prompts to place the software in your preferred directory.
  5. Launch the application from your desktop or start menu shortcut.

🎨 Using Your Stylus

Once you run the application, it creates a local address. Open this address in your iPad web browser. Your browser turns into a digital canvas. Draw your notes or sketches. When you finish, the software saves them directly into your open Emacs session. This happens in real-time as long as the application remains open on your computer.

⚙️ System Requirements

  • Windows 10 or Windows 11.
  • An active installation of GNU Emacs.
  • An iPad or smartphone with a web browser.
  • A stable Wi-Fi connection connecting your computer and tablet.

❓ Frequently Asked Questions

Does this work with Android tablets?

Yes. Any device with a modern web browser and a stylus input works with this software.

Do I need to install a helper package in Emacs?

Yes. The application works best when the corresponding Emacs Lisp package is active, which allows Emacs to listen for incoming drawings.
